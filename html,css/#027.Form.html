• form 태그 자체로는 뷰가 보이지 않는다. 내부의 요소가 존재해야함
  • text input field는 default로 20characters을 담을 수 있는 width를 가진다.
  • label을 누르면 해당하는 input쪽으로 커서가 자동 이동 (혹은 클릭)
  • label 태그의 for 속성값은 input 태그의 id 값과 같아야 한다.
  • name 태그는 꼭 들어가야한다. name 태그가 생략된 값들은 서버로 전송되지 않기 때문이다.
